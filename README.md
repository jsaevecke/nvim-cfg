# Neovim Configuration

Personal Neovim configuration with LSP, completion, Treesitter, Telescope, Harpoon, and AI assistance via Copilot.

---

## Installation

```bash
git clone https://github.com/jsaevecke/neovim-config.git ~/.config/nvim
nvim .
# inside nvim, navigate to lua/pixian/packer.lua and run:
:PackerSync
```

---

## Directory Structure

```
~/.config/nvim/
├── init.lua                   # Entry point — bootstraps the pixian module
│
├── lua/pixian/
│   ├── init.lua               # Module root: loads remap, set, color; filetype detection
│   ├── set.lua                # Editor options (line numbers, tabs, undo, search, ruler…)
│   ├── remap.lua              # Key mappings and autocommands
│   ├── color.lua              # Colorscheme (Tokyo Night)
│   └── packer.lua             # Plugin declarations (packer.nvim)
│
├── after/plugin/              # Plugin-specific configs — loaded automatically after plugins
│   ├── lsp.lua                # LSP servers, Mason, nvim-cmp completion, folds
│   ├── treesitter.lua         # Parser installation + treesitter-context setup
│   ├── telescope.lua          # Fuzzy-finder key mappings
│   ├── harpoon.lua            # File bookmark key mappings
│   ├── trouble.lua            # Diagnostic list + Telescope integration
│   ├── lint.lua               # nvim-lint filetype → linter mapping
│   ├── fugitive.lua           # Git status key mapping
│   └── undotree.lua           # Undo history browser key mapping
│
└── plugin/
    └── packer_compiled.lua    # Auto-generated by PackerSync (do not edit)
```

## Key Mappings Reference

> Leader key: `<Space>`

### General

| Key            | Action                              |
|----------------|-------------------------------------|
| `<leader>pv`   | Open netrw file explorer            |
| `<C-d>` / `<C-u>` | Half-page scroll (cursor centred) |
| `n` / `N`      | Next/prev search result (centred)   |
| `<leader>u`    | Toggle undotree                     |

### LSP

| Key             | Action                        |
|-----------------|-------------------------------|
| `gd`            | Go to definition              |
| `K`             | Hover documentation           |
| `<leader>vrn`   | Rename symbol                 |
| `<leader>vca`   | Code actions                  |
| `<leader>vrr`   | References                    |
| `<leader>vd`    | Open diagnostic float         |
| `[d` / `]d`     | Next/prev diagnostic          |
| `<leader>vws`   | Workspace symbols             |
| `<C-h>`         | Signature help (insert mode)  |
| `<leader>lr`    | Restart LSP server            |
| `zR` / `zM`     | Open / close all folds        |

### Completion (nvim-cmp)

| Key         | Action                  |
|-------------|-------------------------|
| `<C-n>`     | Select next item        |
| `<C-p>`     | Select previous item    |
| `<C-y>`     | Confirm selection       |
| `<C-Space>` | Trigger completion      |

### Telescope

| Key           | Action                              |
|---------------|-------------------------------------|
| `<leader>pf`  | Find files                          |
| `<leader>pg`  | Live grep                           |
| `<leader>ps`  | Grep for entered string             |
| `<C-t>`       | Send Telescope results to Trouble   |

### Harpoon

| Key          | Action                   |
|--------------|--------------------------|
| `<leader>a`  | Mark current file        |
| `<C-e>`      | Toggle quick menu        |
| `<C-h/t/n/s>`| Jump to bookmark 1/2/3/4 |

### Diagnostics (Trouble)

| Key           | Action                        |
|---------------|-------------------------------|
| `<leader>xx`  | Toggle diagnostics            |
| `<leader>xw`  | Workspace diagnostics         |
| `<leader>xd`  | Document diagnostics          |
| `<leader>xq`  | Quickfix list                 |
| `<leader>xl`  | Location list                 |
| `gR`          | LSP references                |

### Go (vim-go)

| Key            | Action                |
|----------------|-----------------------|
| `<leader>t`    | Run current test func |
| `<leader>T`    | Run all tests         |
| `<leader>DS`   | Start debugger        |
| `<leader>DOC`  | Open Go docs          |

### Git

| Key           | Action              |
|---------------|---------------------|
| `<leader>gs`  | Git status (fugitive)|

### Copilot Chat

| Key           | Mode   | Action                          |
|---------------|--------|---------------------------------|
| `<leader>cc`  | Normal | Toggle Copilot Chat window      |
| `<leader>cc`  | Visual | Open chat with selection        |

### Linting

| Key          | Action              |
|--------------|---------------------|
| `<leader>l`  | Manually run linter |
